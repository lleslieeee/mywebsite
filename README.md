# My Choose-Your-Own-Adventure Story

This is a simple **interactive choose-your-own-adventure story** built with HTML, CSS, and JavaScript.  

You make choices at each step, and the story changes based on your decisions. It's a fun way to explore branching narratives and practice web development.

## How to Play

1. Open the website: [Live Story]((https://lleslieeee.github.io/mywebsite/))
2. Read the story text on the screen.
3. Click one of the buttons to make a choice.
4. Follow the story until you reach an ending.

## Features

- Single-page interactive experience
- Styled buttons and simple animations
- Branching story paths

## Notes

- Fully static, no backend required.
- Can be customized by editing `story.js` or adding your own CSS and images.
